# [Grid](http://ionicframework.com/docs/v2/components/#grid)

ionic基于flexbox制作了一套网格框架。

```
<ion-col width-10>This column will take 10% of space</ion-col>
```

列的百分比属性：

- width-10	10%
- width-20	20%
- width-25	25%
- width-33	33.3333%
- width-50	50%
- width-67	66.6666%
- width-75	75%
- width-80	80%
- width-90	90%

用`offset`属性来设置列偏移（例如：`offset-25`）
